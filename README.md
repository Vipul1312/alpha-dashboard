# Alpha Dashboard

A modern, fully responsive admin dashboard built with React, featuring role-based access control, product management, and analytics.

## Features

- Role-based authentication (Admin / User)
- Product listing with search, filters, sorting, and pagination
- Analytics dashboard with charts
- URL state synchronization
- Performance optimizations (debouncing, memo, useMemo, useCallback, lazy loading)
- Fully responsive design

## Tech Stack

- React 18 + Vite
- React Router DOM v6
- Tailwind CSS
- Zustand (state management)
- Recharts
- Axios
